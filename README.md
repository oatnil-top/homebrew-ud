# Homebrew Tap for UnDercontrol CLI (deprecated)

> **⚠️ This Homebrew channel is no longer maintained.**
>
> The UnDercontrol CLI (`ud`) is now distributed via **npm**:
>
> ```bash
> npm install -g @oatnil/ud
> ```

Running `brew install oatnil-top/ud/ud` will print a message pointing to the npm install command above.

## Migrating from a previous brew install

```bash
brew uninstall ud
brew untap oatnil-top/ud
npm install -g @oatnil/ud
```

## About UnDercontrol

UnDercontrol is a task and expense management application. The `ud` CLI provides terminal access to your tasks with a beautiful TUI interface.
