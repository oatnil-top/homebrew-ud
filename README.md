# Homebrew Tap for UnderControl CLI

This is the official Homebrew tap for the UnderControl CLI (`ud`).

## Installation

```bash
brew tap oatnil-top/ud
brew install ud
```

## Upgrade

```bash
brew upgrade ud
```

## Uninstall

```bash
brew uninstall ud
brew untap oatnil-top/ud
```

## About UnderControl

UnderControl is a task and expense management application. The `ud` CLI provides terminal access to your tasks with a beautiful TUI interface.
